À propos de cette application

Cette application a été développée dans le cadre d’un projet universitaire pour gérer les observations d’animaux. Elle permet d’enregistrer des espèces animales avec leur classification biologique, leur statut de conservation, ainsi que les observations associées sur le terrain dans un environnement Symfony + Angular.
